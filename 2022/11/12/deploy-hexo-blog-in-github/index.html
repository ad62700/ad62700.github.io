1.申请一个github 账号

2.登录github 新建一个仓库

首先在 GitHub 新建一个仓库（Repository），名称为 {username}.github.io，注意这个名比较特殊，必须要是 github.io 为后缀结尾的。比如我的用户名是ad62700，那么就新建一个ad62700.github.io的仓库。

3.为仓库配置SSH-Key

因为后期我们更新文章或者提交代码需要有相应的权限才可以，通过用户名和密码不方便且不安全，所以非常有必要配置SSH-Key密钥。

3.1 在本地电脑创建密钥

我电脑是win10, 安装github 后，运行 Git bash

打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key： 将复制内容粘贴
